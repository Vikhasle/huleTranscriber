# huleTranscriber
A custom transcription/summarization model for Human Learning Lab.
The model generates transcripts of audio and creates a summary and view of key moments from the audio.

# Running the transcriber
Install the dependencies detailed in requirements.txt inside an appropriate virtual environment.
Then simply run from the command line:
`python transcriber.py /path/to/audio.mp3`
